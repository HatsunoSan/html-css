Como fazer uma tabela utilizando HTML?

Todas as tabelas - ou pelo menos as mais simlpes - têm a seguinte HIERARQUITA: 

	<!--
        HIERARQUIA DE TABELAS (simples)
        TABLE = tabela
            TABLE ROW = linha de tabela
                TABLE HEADER    =   cabeçalho de tabela
                TABLE DATA      =   dado de tabela
    -->

o W3C, ele coloca como opcional o fechamento das tags <tr> e <td>. PORÉM, se tu não colocar os elementos na hierarquia certa, os dados vão aparecer ou antes, ou depois da tabela.
